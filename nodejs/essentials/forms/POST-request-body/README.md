POST submissions - request body
=================

## Installation

> `npm install`

## Running / Development

### handling text-only form

> Launch server `node server.js`

> Visit your app at http://localhost:3000/signup

> Click "Submit" button to submit form

> Result:

*server*
```
Name: Tom
Email: tom@gmail.com
```
*browser*
```
{
  name: "Tom",
  email: "tom@gmail.com"
}
```
