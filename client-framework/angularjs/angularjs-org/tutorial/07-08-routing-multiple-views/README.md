07-08-routing-multiple-views
============================

## Installation

> `bower install`

## Run Test

> `npm test` run unit-tests

> `ctrl+c` to exit

### Start Server, Run e2e Test, Running App

> `npm start` start server

> `npm run protractor` run e2e-tests

> visit `http://0.0.0.0:8000/app/`
